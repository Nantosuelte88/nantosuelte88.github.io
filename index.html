À propos de Moi

Je suis développeuse web, designer web, et désormais développeuse d'application Python. Passionnée par les nouvelles technologies et le développement de solutions innovantes, j'ai travaillé sur divers projets qui m'ont permis d'acquérir une expertise solide dans plusieurs domaines clés du développement logiciel.

Développez un programme logiciel en Python

Développez un programme logiciel en Python github

  • Création d'un programme permettant de gérer un tournoi d'échecs.
  • Implémentation d'un fichier README.md avec les instructions pour cloner le dépôt, créer et activer l'environnement virtuel, installer les dépendances et exécuter le code.
  • Développement du programme en Python, respectant les spécifications pour être autonome et hors ligne, fonctionnant sous Windows, Mac et Linux.
  • Utilisation de fichiers JSON pour la persistance des données des joueurs et des tournois.
  • Mise en place d'un système pour ajouter des joueurs avec les données : nom, prénom, date de naissance, et identifiant national d'échecs.
  • Gestion des tournois avec les informations suivantes : nom, lieu, date de début et de fin, nombre de tours, numéro du tour actuel, liste des tours, liste des joueurs, et description.
  • Implémentation de la génération des paires de joueurs pour chaque tour en fonction des résultats des matchs précédents, en évitant les matchs identiques.
  • Création d'un système de points pour les matchs : 1 point pour le gagnant, 0 point pour le perdant, et 0,5 point pour un match nul.
  • Mise en place de rapports pour afficher la liste des joueurs par ordre alphabétique, la liste des tournois, les détails d'un tournoi, la liste des joueurs d'un tournoi, et la liste des tours et des matchs d'un tournoi.
  • Sauvegarde et chargement des données à partir de fichiers JSON pour maintenir l'état du programme entre les exécutions.
  • Adoption du modèle de conception MVC (Modèle-Vue-Contrôleur) pour structurer le code de manière maintenable.
  • Utilisation de flake8 avec une longueur de ligne maximale de 119 et génération d'un rapport flake8-html pour assurer la conformité avec les directives PEP 8.

Utilisez les bases de Python pour l'analyse de marché

Utilisez les bases de Python pour l'analyse de marché github

  • Création d'un script Python pour effectuer du web scraping sur le site Books to Scrape.
  • Mise en place d'un fichier README.md détaillant les instructions pour cloner le dépôt, créer et activer l'environnement virtuel, installer les dépendances, et exécuter le code.
  • Développement de la phase 1 du projet : extraction des données d'une page produit spécifique, incluant les informations telles que l'URL de la page produit, le code UPC, le titre, les prix incluant et excluant les taxes, la disponibilité, la description du produit, la catégorie, le taux de critique, et l'URL de l'image.
  • Écriture des données extraites dans un fichier CSV avec les en-têtes appropriés.
  • Réalisation de la phase 2 du projet : extraction des données pour toute une catégorie de livres, en naviguant à travers les pages de pagination si nécessaire.
  • Combinaison des données extraites dans un seul fichier CSV pour la catégorie choisie.
  • Extension du script pour inclure la phase 3 : extraction des informations produit de tous les livres appartenant à toutes les catégories disponibles sur le site Books to Scrape.
  • Écriture des données dans des fichiers CSV distincts pour chaque catégorie de livres.
  • Implémentation de la phase 4 : téléchargement et enregistrement des images de chaque page produit consultée.
  • Utilisation de Python pour automatiser les requêtes HTTP, extraire le HTML des pages web, parser les données avec BeautifulSoup, et gérer l'écriture des données dans des fichiers CSV.
